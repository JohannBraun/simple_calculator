# API.md

## Calculator Module

This module provides basic mathematical operations.

### `add(a, b)`

Adds two numbers together.

**Parameters:**

- `a`: The first number.
- `b`: The second number.

**Returns:** The sum of `a` and `b`.

### `subtract(a, b)`

Subtracts the second number from the first.

**Parameters:**

- `a`: The first number.
- `b`: The second number.

**Returns:** The result of `a - b`.
